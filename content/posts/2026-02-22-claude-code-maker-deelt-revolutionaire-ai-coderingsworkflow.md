TITEL: Claude Code-maker deelt revolutionaire AI-coderingsworkflow
SAMENVATTING: Boris Cherny, maker van Claude Code bij Anthropic, onthulde een workflow waarbij hij vijf AI-agents parallel inzet, wat volgens experts de productiviteit van een enkele ontwikkelaar vergelijkt met die van een kleine afdeling.
TAGS: ai, softwareontwikkeling, productiviteit, mkb

De maker van Claude Code, Boris Cherny, heeft zijn persoonlijke AI-workflow gedeeld, wat wordt gezien als een kantelpunt in softwareontwikkeling. Zijn aanpak: vijf Claude AI-agents parallel inzetten via zijn terminal, waarbij hij de langzaamste maar slimste modelvariant (Opus 4.5) prefereert omdat deze minder correcties nodig heeft. Een gedeeld CLAUDE.md-bestand fungeert als collectief geheugen voor de AI, en geautomatiseerde verificatielussen zorgen dat de code direct werkt. Voor een MKB-ondernemer betekent dit dat slimme inzet van bestaande AI-tools, met focus op kwaliteit en automatisering, productiviteit exponentieel kan verhogen zonder enorme infrastructuurkosten.
