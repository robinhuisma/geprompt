Het AI-codehulpmiddel Goose van Block biedt vergelijkbare functionaliteit als het betaalde Claude Code van Anthropic, maar is gratis, open-source en draait lokaal. Claude Code kost tot 200 dollar per maand en kent gebruiksbeperkingen, wat tot onvrede leidt onder ontwikkelaars. Goose, met ruim 26.100 sterren op GitHub, werkt offline met elke LLM via tools als Ollama, zonder abonnementskosten of cloudafhankelijkheid. Voor een MKB-ondernemer betekent dit een kosteneffectief, privacyvriendelijk alternatief voor AI-ondersteunde softwareontwikkeling, mits de benodigde hardware aanwezig is. Bron: The Verge
