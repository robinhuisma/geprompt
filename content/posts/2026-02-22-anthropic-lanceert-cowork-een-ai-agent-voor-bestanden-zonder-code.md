Anthropic lanceert Cowork, een nieuwe AI-agent voor de Claude Desktop-app. Deze functie, nu in research preview voor Claude Max-abonnees op macOS, geeft de AI toegang tot een aangewezen map op je computer. Claude kan daar bestanden lezen, bewerken en aanmaken voor taken zoals het structureren van een downloadsmap of het maken van een spreadsheet van bonnetjes. De tool is gebouwd op hetzelfde Claude Agent SDK als de coderings-tool Claude Code. Voor MKB'ers kan dit een krachtige productiviteitsboost zijn voor administratieve en organistorische taken, mits men bewust omgaat met de risico's van bestandstoegang. Bron: The Neuron.
