OpenAI heeft nieuwe stemintelligentie-functies aan zijn API toegevoegd, waarmee je als MKB-bedrijf een eigen AI-telefooncentrale kunt bouwen zonder dure maatwerkoplossingen. De API kan spraak herkennen, begrijpen en erop reageren, direct in te zetten voor klantenservice, educatie en creatorplatforms.

## Wat er aan de hand is

OpenAI heeft de mogelijkheden van zijn API uitgebreid met geavanceerde stemintelligentie. Waar de API eerder vooral tekst verwerkte, kan hij nu realtime spraak herkennen, interpreteren en genereren. Volgens TechCrunch zijn de nieuwe functies direct beschikbaar voor ontwikkelaars. Het bedrijf zegt dat de toepassingen verder gaan dan klantenservice – denk aan educatieve tools en interactieve ervaringen voor creators. De lancering past in een bredere trend waarin AI-spraaktechnologie toegankelijker wordt voor kleinere organisaties, zonder dat ze zelf een team van spraaktechnologie-experts hoeven in te huren.

## Wat dit betekent

Voor MKB-bedrijven betekent dit dat een eigen AI-telefooncentrale opeens binnen handbereik komt. Waar je voorheen een dure maatwerkoplossing nodig had van een gespecialiseerd bureau, kun je nu met een API-sleutel een systeem bouwen dat inkomende gesprekken afhandelt, vragen beantwoordt en doorverbindt. Dit is vooral relevant voor sectoren met veel telefonische klantcontacten, zoals webwinkels, zorgpraktijken, installatiebedrijven en administratiekantoren. De drempel om te experimenteren met spraakgestuurde klantenservice wordt aanzienlijk lager, zowel qua kosten als technische complexiteit.

## Hoe je dit kunt toepassen

**Als je een webshop runt met veel telefonische bestellingen of vragen**, kun je overwegen om de API in te zetten voor een automatische telefonische assistent. Een klant belt, zegt ‘ik wil mijn bestelling volgen’, en de AI vraagt om het ordernummer, zoekt de status op en geeft antwoord. Je zou kunnen beginnen met een klein aantal veelgestelde vragen, zoals openingstijden, levertijden en retouren. De API kan de gesprekken ook samenvatten en doorsturen naar je e-mail, zodat je altijd een overzicht hebt.

**Als je een team aanstuurt dat veel telefoontjes krijgt van klanten**, bijvoorbeeld in een installatiebedrijf of een administratiekantoor, dan kun je de API gebruiken om de eerste lijn van klantcontact te automatiseren. De AI kan eenvoudige vragen afhandelen, zoals ‘wanneer komt de monteur?’ of ‘wat zijn jullie tarieven?’. Complexere vragen worden doorgeschakeld naar een medewerker, met een samenvatting van het gesprek. Een mogelijkheid is om de API te koppelen aan je bestaande CRM-systeem, zodat de AI direct klantgegevens kan opvragen en bijwerken.

**Als je in de zorg werkt**, bijvoorbeeld als fysiotherapeut of tandarts, dan kun je de API inzetten voor het plannen van afspraken via de telefoon. Patiënten bellen, de AI vraagt naar hun naam en gewenste datum, en boekt direct een afspraak in je agenda. Je zou kunnen beginnen met een proefperiode waarin alleen nieuwe patiënten worden doorverbonden naar de AI, terwijl bestaande patiënten nog een medewerker aan de lijn krijgen. De praktische toepassing hangt af van jouw situatie, maar de mogelijkheid om telefonische drukte te verminderen is groot.

**Als je een creatorplatform beheert**, zoals een online cursus of een community, dan kun je de API gebruiken om interactieve spraakervaringen te bouwen. Denk aan een AI die vragen van cursisten beantwoordt via spraak, of een stemgestuurde rondleiding door je platform. Een optie is om de API te integreren in een chatbot die ook telefonisch bereikbaar is, zodat gebruikers kunnen kiezen tussen typen en praten.
