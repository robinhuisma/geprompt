De AI-coderingsassistent Claude Code van Anthropic heeft nu een ingebouwde spraakmodus. Dit betekent dat je als ontwikkelaar niet meer alleen hoeft te typen om met de AI te communiceren, maar ook kunt praten. Je kunt vragen stellen, code laten uitleggen of zelfs instructies geven voor het schrijven van nieuwe code, allemaal via gesproken taal. Volgens Anthropic is de modus ontworpen om een meer natuurlijke en snellere interactie mogelijk te maken, vooral voor taken waarbij je je handen vrij wilt hebben of snel een vraag wilt stellen.

## Hoe werkt de spraakmodus in de praktijk?

In plaats van een prompt te typen in de chat, activeer je de microfoon en begin je te praten. De AI luistert, verwerkt je gesproken vraag of opdracht en geeft een antwoord. Dat antwoord kan geschreven code zijn, een uitleg van een codefragment of een suggestie voor het oplossen van een bug. Een mogelijke toepassing is tijdens het coderen, wanneer je met een complex probleem zit en snel een tweede mening wilt. In plaats van te stoppen met typen, vraag je het gewoon hardop. Een andere situatie is het reviewen van code: je kunt een stuk code voorlezen en vragen om een analyse van de veiligheid of efficiëntie.

## Wat zijn de voordelen ten opzichte van typen?

Het grootste voordeel is snelheid voor bepaalde taken. Het formuleren en uitspreken van een vraag gaat vaak sneller dan het typen ervan, vooral voor ontwikkelaars die vloeiend kunnen praten over code. Daarnaast kan het handsfree werken een voordeel zijn, bijvoorbeeld als je tegelijkertijd iets anders op je scherm aan het bekijken bent of notities aan het maken bent. Het kan ook de drempel verlagen voor het stellen van snelle, informele vragen, alsof je een collega naast je hebt zitten. Het is een andere manier van interactie die beter kan passen bij je persoonlijke workflow.

## Waar moet je op letten?

De spraakmodus is een nieuwe feature en zal, zoals bij alle AI-tools, zijn beperkingen hebben. De nauwkeurigheid van spraakherkenning in een technische context, vol met jargon en specifieke termen, is een punt van aandacht. Daarnaast is het een tool van een specifieke aanbieder, Claude Code, en geen universele standaard. De effectiviteit hangt af van hoe goed de onderliggende AI-model je gesproken intentie begrijpt en vertaalt naar een nuttige actie. Het is een extra interface, geen vervanging voor de bestaande geschreven chat.

## Hoe kun je dit vandaag toepassen?

Als je Claude Code gebruikt, is een mogelijkheid om de nieuwe spraakmodus eens te proberen tijdens een code-review sessie. Je zou een stuk code kunnen voorlezen en hardop kunnen vragen naar potentiële verbeterpunten of veiligheidsissues. Een andere suggestie is om het te gebruiken voor het snel opzoeken van syntax of het laten uitleggen van een foutmelding zonder je handen van het toetsenbord te halen.
