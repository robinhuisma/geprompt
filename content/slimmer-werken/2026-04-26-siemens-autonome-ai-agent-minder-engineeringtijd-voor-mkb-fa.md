Siemens heeft op de Hannover Messe een AI-agent gepresenteerd die zelfstandig engineeringtaken uitvoert, zoals het schrijven van code en configureren van systemen. Voor MKB-fabrikanten betekent dit dat repetitief engineeringwerk tot vijf keer sneller kan worden gedaan, zonder dat je zelf een AI-specialist in dienst hoeft te nemen.

## Wat er aan de hand is
Siemens heeft de Eigen Engineering Agent gelanceerd, een AI-oplossing die niet alleen advies geeft, maar daadwerkelijk taken uitvoert binnen industriële automatiseringsomgevingen. Volgens het bedrijf kan de agent automatiseringsprojecten doorgronden, code schrijven, systemen configureren en resultaten valideren tot aan vooraf gestelde prestatie-eisen is voldaan. De agent is getest bij meer dan honderd bedrijven in negentien landen, waaronder Andritz Metals uit Oostenrijk en Prism Systems uit de VS. De oplossing is commercieel beschikbaar voor gebruikers van Siemens’ TIA Portal, dat wereldwijd meer dan zeshonderdduizend gebruikers telt. Siemens investeert een miljard euro in industriële AI en heeft meer dan vijftienhonderd AI-specialisten in dienst.

## Wat dit betekent
Voor Nederlandse MKB-fabrikanten die werken met Siemens-automatisering opent dit de deur naar aanzienlijke tijdsbesparing. Siemens stelt dat de agent taken twee tot vijf keer sneller uitvoert dan handmatige workflows, zonder concessies aan nauwkeurigheid. In specifieke toepassingen zou dit kunnen leiden tot vijftig procent hogere engineeringefficiëntie. Het tekort aan gekwalificeerde engineers is in de industrie voelbaar, en deze AI-agent kan repetitieve taken overnemen, zoals PLC-programmering, HMI-visualisaties en apparaatconfiguratie. Dit betekent dat kleinere bedrijven zonder grote engineeringafdelingen complexere projecten kunnen aanpakken, of bestaande projecten sneller kunnen afronden.

## Hoe je dit kunt toepassen
**Als je een klein productiebedrijf runt met één of twee engineers.** Je zou de Eigen Engineering Agent kunnen inzetten voor standaard PLC-programmering. In plaats van dat je engineer uren besteedt aan het schrijven van code voor een nieuwe transportband, kan de agent dit binnen een fractie van de tijd doen. Je engineer kan zich dan richten op complexere vraagstukken, zoals optimalisatie van het productieproces. Een mogelijkheid is om te beginnen met een pilotproject, zoals het automatiseren van een eenvoudige machine, om te ervaren hoe de agent werkt binnen jouw omgeving.

**Als je een machinebouwer bent die meerdere projecten tegelijk draait.** De agent kan worden gebruikt voor het configureren van systemen en het schrijven van HMI-visualisaties. Overweeg om de agent in te zetten voor de repetitieve delen van elk project, zoals de standaard veiligheidslogica of de basisvisualisatie. Dit kan de doorlooptijd van projecten verkorten, waardoor je meer orders kunt aannemen zonder extra personeel. Je zou kunnen testen of de agent de consistentie in je code verhoogt, wat fouten bij oplevering vermindert.

**Als je een engineer bent die worstelt met een tekort aan tijd.** Je zou de agent kunnen gebruiken als een assistent die voorstellen doet voor code of configuraties. In plaats van alles zelf te schrijven, geef je de agent een specificatie en laat je hem een eerste versie maken. Jij controleert en past aan waar nodig. Dit kan je helpen om projecten sneller af te ronden, zonder dat je concessies doet aan kwaliteit. Een optie is om de agent in te zetten voor taken die je als vervelend of tijdrovend ervaart, zoals het documenteren van configuraties.

**Als je bedrijf nog niet werkt met Siemens TIA Portal.** De agent is alleen beschikbaar voor TIA Portal-gebruikers. Als je overweegt over te stappen naar Siemens-automatisering, kan de beschikbaarheid van deze AI-agent een extra argument zijn. Je zou kunnen informeren bij Siemens of er mogelijkheden zijn voor een proeflicentie of demo. De praktische toepassing hangt af van jouw situatie, maar het is de moeite waard om te onderzoeken of de agent past bij jouw automatiseringsbehoeften.
