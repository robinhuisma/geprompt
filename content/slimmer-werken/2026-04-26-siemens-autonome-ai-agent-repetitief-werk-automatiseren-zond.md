Siemens heeft een AI-agent gelanceerd die zelfstandig engineeringtaken in industriële automatisering uitvoert, zoals het schrijven van code en configureren van systemen. De agent is getest bij meer dan honderd bedrijven en belooft repetitief werk twee tot vijf keer sneller af te handelen dan handmatige workflows. Voor MKB-bedrijven zonder eigen AI-specialisten opent dit de mogelijkheid om complexe automatiseringstaken uit te besteden aan een systeem dat geen uitleg nodig heeft.

## Wat er aan de hand is

Siemens heeft op de Hannover Messe de Eigen Engineering Agent gepresenteerd. Dit is een AI-oplossing die zelfstandig engineeringtaken uitvoert binnen industriële automatisering. Volgens Siemens verschilt de agent van bestaande AI-tools en copilots doordat hij niet alleen advies geeft, maar taken daadwerkelijk uitvoert in echte engineeringomgevingen. De agent kan automatiseringsprojecten doorgronden, code schrijven, systemen configureren en resultaten valideren tot vooraf vastgestelde prestatie-eisen zijn gehaald.

De agent is getest bij meer dan honderd bedrijven in negentien landen, waaronder Andritz Metals uit Oostenrijk, Casmt uit China en Prism Systems uit de Verenigde Staten. In de pilots werd de AI ingezet voor PLC-programmering, HMI-visualisaties en apparaatconfiguratie. De agent is commercieel beschikbaar voor gebruikers van Siemens’ TIA Portal, dat wereldwijd meer dan zeshonderdduizend gebruikers telt. Siemens meldt dat de agent taken twee tot vijf keer sneller uitvoert dan handmatige workflows, wat in specifieke toepassingen kan leiden tot vijftig procent hogere engineeringefficiëntie.

## Wat dit betekent

Voor MKB-bedrijven in de maakindustrie of procesautomatisering betekent dit dat repetitieve engineeringtaken niet langer een bottleneck hoeven te zijn. Het tekort aan gekwalificeerde engineers is voelbaar, en deze AI-agent kan routinematig werk overnemen zonder dat je zelf AI-experts in dienst hoeft te hebben. De agent werkt binnen de bestaande Siemens-omgeving, dus je hoeft geen nieuwe systemen te leren.

De impact is het grootst voor bedrijven die veel standaardconfiguraties doen, zoals het programmeren van PLC’s of het maken van HMI-visualisaties. Dit zijn taken die vaak veel tijd kosten en waar consistentie belangrijk is. Als de agent deze taken twee tot vijf keer sneller kan doen, levert dat directe tijdwinst op voor je eigen engineers, die zich dan kunnen richten op complexere vraagstukken.

## Hoe je dit kunt toepassen

**Als je een productiebedrijf runt met een beperkt engineeringteam.** Je zou de Eigen Engineering Agent kunnen inzetten voor het schrijven van PLC-code voor nieuwe machines of het configureren van aandrijvingen. In plaats van dat je engineer uren besteedt aan het schrijven van standaardcode van nul, geef je de agent de specificaties en laat je hem het werk doen. Je engineer controleert alleen de output. Een mogelijkheid is om te beginnen met een pilot op een klein project, zoals het herprogrammeren van een bestaande productielijn.

**Als je een systeemintegrator bent die voor meerdere klanten werkt.** Overweeg om de agent te gebruiken voor het standaardiseren van configuraties. Veel klanten vragen om vergelijkbare oplossingen, maar elke keer moet je opnieuw configureren. De agent kan op basis van een template de configuratie genereren en valideren. Je zou kunnen beginnen met het automatiseren van de meest voorkomende configuratie die je uitvoert, bijvoorbeeld voor pompen of transportbanden.

**Als je in de voedingsmiddelenindustrie werkt en veel herhalende validaties moet doen.** De agent kan resultaten valideren tegen vooraf gestelde eisen. In plaats van dat een engineer handmatig controleert of alle parameters kloppen, laat je de agent dit doen. Een optie is om de agent in te zetten bij het configureren van nieuwe verpakkingslijnen, waarbij elke lijn dezelfde basisconfiguratie heeft maar kleine variaties kent.

**Als je een machinebouwer bent die worstelt met doorlooptijden.** De agent kan helpen om de tijd tussen order en oplevering te verkorten. Je zou de agent kunnen gebruiken om direct na een order de basisconfiguratie te genereren, zodat je engineer alleen nog de maatwerkonderdelen hoeft te doen. Een mogelijkheid is om de agent in te zetten voor het genereren van HMI-visualisaties, een taak die vaak veel handwerk kost.
