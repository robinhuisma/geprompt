Siemens heeft op de Hannover Messe een AI-agent gepresenteerd die zelfstandig engineeringtaken in industriële automatisering kan uitvoeren. De Eigen Engineering Agent schrijft code, configureert systemen en valideert resultaten, zonder dat een mens er continu aan te pas komt. Voor Nederlandse MKB-fabrikanten met een tekort aan engineers kan dit betekenen dat repetitief werk twee tot vijf keer sneller wordt afgehandeld.

## Wat er aan de hand is

Siemens heeft de Eigen Engineering Agent gelanceerd, een AI-oplossing die taken in industriële automatisering volledig autonoom uitvoert. Volgens het bedrijf verschilt deze agent van bestaande AI-tools doordat hij niet alleen advies geeft, maar daadwerkelijk code schrijft, systemen configureert en resultaten valideert, net zolang tot vooraf vastgestelde prestatie-eisen zijn gehaald. De agent is getest bij meer dan honderd bedrijven in negentien landen, waaronder Andritz Metals uit Oostenrijk, Casmt uit China en Prism Systems uit de Verenigde Staten. De oplossing is commercieel beschikbaar voor gebruikers van Siemens’ TIA Portal, dat wereldwijd meer dan zeshonderdduizend gebruikers telt. Siemens investeerde vorig jaar een miljard euro in industriële AI en heeft meer dan vijftienhonderd AI-specialisten in dienst.

## Wat dit betekent

Voor Nederlandse MKB-fabrikanten die werken met Siemens-automatisering is dit een concrete uitbreiding van hun gereedschapskist. De agent richt zich op repeterende engineeringtaken zoals PLC-programmering, HMI-visualisaties en apparaatconfiguratie. Juist die taken zijn tijdrovend en worden vaak door schaarse engineers gedaan. Siemens claimt dat de agent taken twee tot vijf keer sneller uitvoert dan handmatige workflows, zonder concessies aan nauwkeurigheid. In specifieke toepassingen zou dit kunnen leiden tot vijftig procent hogere engineeringefficiëntie. Voor een klein ingenieursbureau of een productiebedrijf met een beperkt team betekent dit dat standaardwerk sneller klaar is, waardoor engineers zich kunnen richten op complexere vraagstukken. De prijs van de agent wordt niet genoemd in de aankondiging; die hangt af van de licentievorm binnen het Siemens Xcelerator-portfolio.

## Hoe je dit kunt toepassen

**Als je een klein ingenieursbureau runt dat PLC-programmering uitvoert voor klanten in de maakindustrie.** Je zou de Eigen Engineering Agent kunnen inzetten voor standaardprogrammeerwerk, zoals het schrijven van code voor eenvoudige besturingslogica. De agent doorloopt het project, schrijft de code en valideert of de prestatie-eisen zijn gehaald. Dat bespaart uren handmatig werk per project. Een mogelijkheid is om de agent eerst te testen op een niet-kritisch project, zodat je ervaart hoe de output zich verhoudt tot jouw kwaliteitsstandaarden.

**Als je een productiebedrijf bent met een eigen technische dienst die machines configureert.** Overweeg om de agent in te zetten voor het configureren van nieuwe apparaten of het aanpassen van bestaande HMI-schermen. De agent kan variaties doorrekenen en configuraties genereren die voldoen aan de specificaties. Je technische dienst hoeft dan alleen de resultaten te controleren in plaats van alles vanaf nul op te bouwen. Dat kan de doorlooptijd van een machine-ombouw verkorten van dagen naar uren.

**Als je worstelt met het vinden van gekwalificeerde engineers voor je automatiseringsprojecten.** De agent kan een deel van het repetitieve werk overnemen, waardoor je met een kleiner team meer projecten kunt doen. Je zou kunnen beginnen met één project waar de agent de volledige engineeringcyclus doorloopt, van analyse tot validatie. Op basis van die ervaring kun je bepalen voor welke type projecten de agent geschikt is en waar je liever handmatig werkt.

**Als je als zelfstandig engineer werkt en je productiviteit wilt verhogen.** De agent kan fungeren als een virtuele assistent die de standaardcode voor je schrijft. Je zou de agent een specificatie kunnen geven en de gegenereerde code kunnen reviewen en aanpassen. Dat kan je helpen om meer projecten in dezelfde tijd af te ronden, zonder dat je concessies doet aan de kwaliteit van het eindresultaat.
