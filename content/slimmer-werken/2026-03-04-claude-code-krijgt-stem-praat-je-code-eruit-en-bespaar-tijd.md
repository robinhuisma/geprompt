De AI-coderingsassistent Claude Code van Anthropic heeft nu een stemmodus. Dit betekent dat je niet meer alleen hoeft te typen om code te schrijven of problemen op te lossen, maar dat je ook met de assistent kunt praten. Het is een stap om het ontwikkelproces natuurlijker en mogelijk sneller te maken, vooral in situaties waar je handen vrij moeten zijn of waar een gesprek een betere manier is om een complex probleem uit te leggen.

## Wat de stemmodus praktisch betekent

In plaats van prompts te typen, kun je nu je vraag of instructie uitspreken. Claude luistert en reageert met gesproken antwoorden en genereert de gevraagde code. Volgens Anthropic is dit handig tijdens het brainstormen, bij het uitleggen van een ingewikkelde bug, of wanneer je snel een codefragment nodig hebt terwijl je aan iets anders werkt, zoals het configureren van een server of het reviewen van een ontwerp. Het verandert de interactie van een typ-sessie in een dialoog.

## De mogelijke impact op je workflow

Voor een individuele ontwikkelaar of een klein team kan dit een manier zijn om sneller van idee naar uitvoering te gaan. Je bespreekt een logisch probleem hardop, en Claude kan ter plekke de syntaxis en structuur leveren. Het kan ook helpen bij het documenteren van code: je legt uit wat een functie moet doen, en Claude schrijft de code met bijbehorende comments. In een MKB-context, waar tijd schaars is, kan dit de snelheid van prototyping en het oplossen van dagelijkse technische problemen verhogen.
