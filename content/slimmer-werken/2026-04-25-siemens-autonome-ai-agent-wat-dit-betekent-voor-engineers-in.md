Een autonome AI-agent van Siemens kan nu zelfstandig industriële automatiseringsprojecten uitvoeren, van het schrijven van code tot het configureren van systemen. Voor het Nederlandse MKB in de maakindustrie en engineering betekent dit dat repeterende taken twee tot vijf keer sneller kunnen worden uitgevoerd, zonder dat er extra engineers nodig zijn. De agent is direct beschikbaar voor gebruikers van het TIA Portal-platform.

## Wat er aan de hand is

Siemens heeft op de Hannover Messe de Eigen Engineering Agent gepresenteerd, een AI-oplossing die engineeringtaken op het gebied van industriële automatisering zelfstandig uitvoert. Volgens het bedrijf verschilt deze agent van bestaande AI-tools doordat hij niet alleen advies geeft, maar taken daadwerkelijk uitvoert binnen echte engineeringomgevingen. De agent kan automatiseringscode schrijven, systemen configureren en resultaten valideren, net zo lang tot voorafgestelde prestatie-eisen zijn gehaald. De oplossing is getest bij meer dan honderd bedrijven in negentien landen, waaronder Andritz Metals, Casmt en Prism Systems. De agent is commercieel beschikbaar voor gebruikers van Siemens’ TIA Portal, dat wereldwijd meer dan zeshonderdduizend gebruikers telt. De introductie past binnen de bredere AI-strategie van Siemens, dat vorig jaar een investering van een miljard euro in industriële AI aankondigde.

## Wat dit betekent

Voor het Nederlandse MKB in de industriële automatisering en machinebouw is dit een concrete verschuiving. Waar AI tot nu toe vooral ondersteunend was, neemt deze agent nu zelfstandig taken over. Siemens claimt dat de agent taken twee tot vijf keer sneller uitvoert dan handmatige workflows, zonder concessies aan nauwkeurigheid. In specifieke toepassingen zou dit kunnen resulteren in tot vijftig procent hogere engineeringefficiëntie. Dit is met name relevant voor bedrijven die kampen met een tekort aan gekwalificeerde engineers. De agent richt zich op repeterende werkzaamheden zoals PLC-programmering, HMI-visualisaties en apparaatconfiguratie. Voor een klein ingenieursbureau met drie engineers kan dit betekenen dat hetzelfde werk in de helft van de tijd wordt gedaan, of dat er zonder extra personeel meer projecten kunnen worden aangenomen.

## Hoe je dit kunt toepassen

**Als je een klein ingenieursbureau runt dat PLC-programmering uitbesteedt**, overweeg om de Eigen Engineering Agent in te zetten voor standaard programmeertaken. Je zou kunnen beginnen met een pilotproject waarin de agent een eenvoudige besturingskast configureert. De agent werkt binnen het TIA Portal, dus als je dat al gebruikt, is de drempel laag. Je kunt de agent instellen op jouw specifieke prestatie-eisen, zoals cyclustijden of veiligheidsnormen. Een mogelijkheid is om eerst de repeterende 80 procent van het werk door de agent te laten doen en alleen de complexe uitzonderingen zelf te behandelen.

**Als je een machinebouwer bent die worstelt met doorlooptijden**, overweeg om de agent in te zetten voor HMI-visualisaties. Dit is vaak een tijdrovend onderdeel van een project. Je zou de agent een basisscherm kunnen laten genereren op basis van een functioneel ontwerp, waarna jij alleen nog de details aanpast. De agent kan net zo lang itereren tot de vooraf bepaalde prestatie-eisen zijn gehaald, wat betekent dat je niet steeds handmatig hoeft te controleren. Dit kan de doorlooptijd van een machineproject met dagen verkorten.

**Als je een technisch directeur bent die schaarse engineers optimaal wil inzetten**, overweeg om de agent te gebruiken voor validatie en testwerk. De agent kan resultaten valideren en fouten opsporen voordat een engineer er tijd aan besteedt. Je zou kunnen afspreken dat alle nieuwe PLC-code eerst door de agent wordt gecontroleerd op consistentie en fouten, voordat een senior engineer hem beoordeelt. Dit verhoogt de kwaliteit en vermindert het aantal revisierondes. Een optie is om de agent in te zetten als een extra kwaliteitscontroleur die nooit moe wordt.

**Als je een onderhoudsbedrijf bent dat installaties configureert op afstand**, overweeg om de agent te gebruiken voor standaard configuratietaken. De agent kan op basis van een vragenlijst of een eenvoudige input een volledige configuratie genereren, die jij alleen nog hoeft te controleren. Dit bespaart reistijd en maakt het mogelijk om meer projecten per dag te doen. Je zou kunnen beginnen met één type installatie dat vaak terugkomt en de agent laten leren van jouw eerdere configuraties.
