De AI-coderingsassistent Claude Code van Anthropic krijgt een spraakmodus. Dit betekent dat ontwikkelaars straks niet alleen kunnen typen, maar ook tegen hun AI-assistent kunnen praten om code te schrijven, te debuggen of te documenteren. Het is een stap die de interactie met codeertools natuurlijker en mogelijk sneller kan maken, vooral voor bepaalde taken.

## Wat betekent dit voor je workflow?

De kern van de aankondiging is dat je straks een gesprek kunt voeren met de tool die je helpt bij het programmeren. In plaats van prompts te typen, kun je je vraag of opdracht uitspreken. Volgens Anthropic kan dit handig zijn tijdens het brainstormen, wanneer je snel een concept wilt uitwerken zonder je handen van het toetsenbord te halen. Het kan ook helpen bij het documenteren van code, waar spraak vaak sneller gaat dan typen. Voor ontwikkelaars die veel multitasken of tijdens het coderen ook andere zaken moeten regelen, kan dit een productiviteitswinst opleveren.

## Waar ligt de praktische waarde?

De echte waarde zit hem niet in het volledig vervangen van het toetsenbord, maar in het aanvullen ervan voor specifieke momenten. Denk aan het snel opzetten van een boilerplate voor een nieuwe functie, het stellen van een verhelderende vraag over een complex stuk code terwijl je ernaar staart, of het genereren van commentaar en documentatie. Het kan de flow minder onderbreken, omdat je niet hoeft te schakelen tussen denken en typen. Voor Nederlandse ontwikkelteams die werken in agile of snelle ontwikkelcycli, kan dit soort tooling helpen om prototypes nog sneller van de grond te krijgen.

## Hoe verhoudt dit zich tot andere tools?

Claude Code is niet de eerste die met spraak experimenteert. Andere geïntegreerde ontwikkelomgevingen (IDE's) en AI-assistenten hebben ook verkennende features op dit gebied. De stap van Anthropic is vooral significant omdat het een native, geïntegreerde modus wordt in een tool die specifiek is gericht op coderen, in plaats van een algemene AI-chatbot. De effectiviteit zal afhangen van hoe goed de AI gesproken, technische instructies begrijpt en vertaalt naar precieze, werkende code.
